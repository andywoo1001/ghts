ZeroMQ 4의 Go언어 바인딩인 zmq4는 C언어 컴파일 과정이 필요하기에,
일반적인 Go언어 모듈 설치보다 추가적인 설치준비 과정이 필요하다.

1. Ruby DevKit 설치 :
- 무료 C언어 컴파일러인 GCC의 윈도우용 버전.
- ZeroMQ C언어 코드를 컴파일 하는 데 필요한 기능이 모두 갖추어져 있음. 
- http://rubyinstaller.org/downloads/
- RubyInstallers 가 아님. Ruby DevKit Installer임.
- Other Useful Downloads 밑에 있는 DEVELOPMENT KIT를 설치해야 함.
- 

2. ZeroMQ 4 소스코드 설치 :
- cd <적절한 작업 디렉토리>
- git clone https://github.com/zeromq/zeromq4-x
- cd zeromq4-x\builds\mingw32
- copy Makefile.mingw32 Makefile
- make

4. ZeroMQ 설치
다음 3개의 파일을 PATH가 걸린 디렉토리에 복사.
- libzmq.dll
- libgcc_s_sjlj-1.dll
- libstdc++-6.dll

5. Go언어 바인딩 zmq4 설치
cmd.exe에서 실행
- <ruby_devkit 디렉토리>\devkitvars.bat
- set C_INCLUDE_PATH=<zmq 소스코드 디렉토리>\zeromq4-x\include
- set LIBRARY_PATH=<zmq 소스코드 디렉토리>\zeromq4-x\builds\mingw32    (libzmq.dll 이 있는 디렉토리)
- go get -u github.com/pebbe/zmq4

위에 나온 모든 과정을 다음과 같이 할 수 있다.
1. Go언어 설치
2. go get github.com/ghts/ghts
3. %GOPATH%\src\github.com\ghts\ghts\bin\zmq4_install.bat
