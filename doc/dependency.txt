Go Dependency

1. Go binding for ZeroMQ
> go get github.com/pebbe/zmq3

2. Encoding conversion for Korean language
> go get github.com/suapapa/go_hangul/encoding/cp949

3. go-decimal
> go get github.com/landjur/go-decimal


Python Dependency

1. Python binding for ZeroMQ
> pip install pyzmq

2. Named constants
> pip install named_constants
